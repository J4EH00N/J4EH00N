- 🔭 브라우저 확장 프로그램 공급망 보안 분석 시스템 연구 (팀장)
- 🔐 AppSec, 취약점 분석(BOF/ROP 등 시스템 해킹), 네트워크 보안
- 🧠 정적/동적 분석 + RAG/LLM 기반 보안 판단 지원 설계
- ✍️ 학습 내용을 블로그·Writeup으로 정리해 공유
- 🎯 **Open to:** 보안 엔지니어링 / 위협 인텔리전스 / AppSec 인턴·신입 포지션
